# 宠物零食品牌招商 PPT 代码参考

## 项目定位

- 项目是一个纯静态中文宠物零食品牌站，前台面向消费者展示天然宠物零食，后台面向运营演示订单、营业额和商品管理。
- 品牌当前线上页面标题为“岚禾小食 | 天然宠物零食”，主张是“单一原料 · 低温慢制 · 日常奖励”。
- 页面核心表达是“给毛孩子一口干净、真实的好味道”，强调可溯源肉源、温和配方、日常训练、陪伴和挑嘴时刻。

## 前台商品与消费场景

- 商品数据集中维护在 `script.js`，字段包括名称、宠物分类、食品分类、价格、销量、规格、图片、标签、成分、适用建议、适口性和简介。
- 默认商品覆盖鸡肉、冻干、洁齿、训练奖励等场景，支持猫猫、狗狗和猫狗通用筛选。
- 前台提供热销榜、商品卡片、商品详情弹窗、购物车抽屉和本地模拟下单流程。
- 商品详情强调主要成分、适用建议和适口性，适合招商 PPT 用作“产品卖点标准化表达”。

## 账号、购物车与订单

- 站点包含登录、注册和游客入口；游客可以浏览与加购，购买时引导登录。
- 登录后可管理多个收货地址，结算前可选择或新增地址。
- 确认购买后订单写入 `localStorage["pawNaturalOrders"]`，订单包含订单号、时间、客户、地址、商品明细、金额和状态。

## 管理后台能力

- 管理后台文件包括 `admin.html` 和 `admin.js`，定位为公开演示型后台，不做真实权限校验。
- 后台展示经营概览：总营业额、订单数、售出件数和销量冠军。
- 后台展示订单列表、商品销量排行、商品销售明细，并支持新品上架。
- 新品写入 `localStorage["pawNaturalProducts"]` 后，前台会合并读取“默认商品 + 后台新品”，新品可参与筛选、详情、购物车和下单。

## 可用于招商叙事的商业要点

- 品牌价值：天然、干净、短配方、低温慢制、中文成分标注。
- 产品矩阵：原切肉、冻干、洁齿、训练奖励，覆盖日常奖励、挑嘴加餐、口腔护理和训练互动。
- 运营能力：前台选购链路完整，后台能演示订单与销售统计，适合向渠道商展示“产品 + 数字化运营”的组合能力。
- 招商逻辑：以高端天然宠物零食为切入点，面向宠物门店、社区团购、内容电商、私域团长和区域代理开展合作。
- 合作卖点：商品容易标准化陈列，场景化文案清晰，复购场景稳定，后台数据可辅助选品和补货。
